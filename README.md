## Installing ✨

- Download as ZIP;
- Open Blender;
- Edit -> Preferences;
- Add-ons -> Install;
- Select downloaded ZIP;
- Typo in search "soulworker";
- Mark result;
- File -> Import -> SoulWorker (.model).
